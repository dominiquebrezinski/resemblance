Class Resemblance::ResultsClusterer
In: resemblance.rb
Parent: Object

Uses K-Means clustering to segment the result set generated by evaluate_text_against_corpus. The general use-case is to identify results from the corpus that come near or above a certain R threshold.

Methods

min_and_max_r   new   results_over_r_threshold  

Public Class methods

new(result_set, cluster_count = 5)
result_set:the result set from evaluate_text_against_corpus or any set in the form [ [name, R],…]
cluster_count:the number of clusters to segment the results into. default is 5

Public Instance methods

min_and_max_r()

Returns an array containing the centroid‘s R values for the smallest and largest, with regard to R, clusters.

results_over_r_threshold(r_threshold)

Select only the results from clusters where the centroid‘s R is greater than the given threshold.

r_threshold:a float between 0.0 and 1.0. for short text chunks, under 300 words, and Ngraphs a value of 0.2 is a good start. when using Ngrams try 0.03.
