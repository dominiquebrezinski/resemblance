Encompasses the set of Ngraphs for a given text and operations over the set.

Public Class methods

new(text, options = {})
text:the text for evaluation
options:a hash of options to control text preparation, tokenization and the length of the Ngraph. :character_removal_set must be a RegExp that matches the characters to be removed from the text prior to tokenization. The default is /[^a-zA-Z0-9 ]/. :token_separator must be a String or RegExp that identifies the token (word) separator in the prepared text. The default is /\s*/ which has the effect of removing whitespace and splitting the text into individul characters. :n is the length of the Ngraphs and defaults to 3.
